# Simple Chatbot (Python)

This project is a simple chatbot built using Python and Jupyter Notebook.

## Features
- Basic conversational chatbot
- Multiple versions (v1, v2)
- Uses simple logic / NLP techniques

## Tech Stack
- Python
- Jupyter Notebook

- Structuring AI-related projects
